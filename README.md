# ShopGUI

## Описание: 
Полноценный магазин предметов с графическим интерфейсом (меню сундука).
Цены настраиваются в конфиге. Защита от дюпов. Звуковое сопровождение покупок.

## Команды:
/shop — Открыть меню магазина.
